# Internship Registration Portal

A secure web-based internship registration system built using PHP and MySQL.

## Features
- Client-side validation
- Server-side validation
- Password hashing
- File upload security (PDF only)
- SQL Injection prevention (Prepared Statements)
- XSS protection

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Install WAMP/XAMPP
2. Place project in www folder
3. Import database in phpMyAdmin
4. Run on localhost
